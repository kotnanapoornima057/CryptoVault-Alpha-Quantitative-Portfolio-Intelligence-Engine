# 🚀 CryptoVault Alpha – Quantitative Portfolio Intelligence Engine

CryptoVault Alpha is a full-stack cryptocurrency portfolio management platform that enables users to securely manage digital assets, track portfolio performance, analyze tax liabilities, assess investment risk, and interact with an AI-powered financial assistant.

The application provides real-time portfolio analytics, multiple cost-basis accounting strategies, tax reporting, AI-based portfolio insights, and an intuitive modern user interface.


📌 Features

🔐 User Authentication
- Secure user registration and login using JWT authentication.
- Password encryption using bcrypt.
- Protected API routes.

📊 Portfolio Dashboard
- Live portfolio overview.
- Total investment tracking.
- Current portfolio value.
- Unrealized Profit/Loss calculation.
- Asset allocation table.
- Portfolio allocation chart.
- Portfolio performance trend chart.

💼 Ledger Operations
- Add cryptocurrency buy/sell transactions.
- Transaction history management.
- Automatic portfolio updates.

📈 Performance Analytics
- Portfolio growth analysis.
- Investment performance visualization.
- Asset-wise performance tracking.

⚠️ Risk Management
- Portfolio risk assessment.
- Diversification analysis.
- Investment risk monitoring.

💰 Tax Analytics
- FIFO (First In First Out)
- LIFO (Last In First Out)
- HIFO (Highest In First Out)
- Realized Capital Gain calculation.
- Tax liability estimation.
- Disposal lot analysis.
- PDF Tax Report Export.

🤖 AI Advisor
- AI-generated portfolio insights.
- Personalized investment recommendations.
- Performance analysis.
- Risk analysis.
- Tax-related suggestions.

💬 AI Chat Assistant
- Chat with an AI financial assistant.
- Conversation history.
- Delete previous chats.
- Speech-to-text input.
- Text-to-speech response.
- Copy AI responses.
- Like/Dislike feedback.

🌙 User Experience
- Dark Mode support.
- Responsive modern UI.
- Toast notifications.
- Beautiful dashboard design.
